## 1. 핵심 가설 (확정)

보이니치 문서는 **14~15세기 북이탈리아(파두아 권역) 수도원 전통의 식물약초서**로,  
**라틴어 어근 축약 + 문법 어미 보존** 방식으로 압축 기록된 식물 해부학·처방 문서.

### 문서 구조
| 섹션 | 폴리오 | 정체 |
|---|---|---|
| 식물 | f1~f66 | 각 식물의 구조+특성+환경 기술 |
| 천문 | f70v~f73v | Homo Signorum 식물학적 변형 = 계절별 약초 채취 달력 |
| 생물 | f74~f84 | Fasciculus Medicinae 정맥도 변형 = 수액 흐름 의인화 |
| 약학(뿌리도감) | f87~f96v | 식물 단면 부위 도감 |
| 약학(처방) | f99~f102 | 뿌리/줄기 단면 해부도 레이블 (항아리 아님) |
| 레시피 | f103~f116 | [부위]+[처리동사(am)] 처방문, 줄 단위 독립 처방 |

## 2. 저자의 세 가지 핵심 아이디어

이 세 가지 아이디어는 저자(김욱, 독립 연구자)로부터 출발했다.

**아이디어 1 — f99~f102 약학 섹션을 식물 단면도로 파악**  
기존에 약통(apothecary jar)으로 해석되던 f99~f102의 삽화를 식물 줄기/뿌리의 횡단면 해부도로 재해석했다. 동심원 층위 구조, 유기적 불규칙 외곽선, 갈색(목질)/초록(초본) 색상 코딩이 도자기 용기가 아닌 단면 해부도와 일치한다는 관찰에서 출발했다.

**아이디어 2 — 생물 섹션 목욕 여성을 수관/체관 흐름 의인화로 파악**  
목욕하는 여성 인물들을 식물의 물관(xylem)·체관(phloem) 수액 이동을 의인화한 표현으로 재해석했다. Fasciculus Medicinae의 정맥도(Vein Man, 1491)를 인체 대신 식물에 적용한 도상학적 변형으로 본다.

**아이디어 3 — 천문 섹션을 식물 계절 채취 달력으로 파악**  
조디악 도표를 Homo Signorum 전통의 식물학적 변형으로 제안했다. 12궁이 인체 부위 대신 식물 해부학 층위(cortex, duramen, alburnum)를 지배하며, 계절별 최적 채취 시기를 지정하는 Tacuinum Sanitatis 달력 구조다.

## 5. 어순 규칙 3개 (v20, 통계 검증)

**규칙 1** — k(coquitur)가 문장 첫머리에 올 수 있다 (처방문의 3.8%). am/om 단독으로 처방 종결 가능.

**규칙 2** — 특성어(PROP)는 수식 대상 구조어 바로 뒤에 후치 (93.1% 준수). 반례 유형: A(문두 독립 명제), B(FUNC 뒤 새절), C(특성어끼리 수식).

**규칙 3** — 환경어(QUAL)는 문장 가장자리에만 출현 (82.3% 준수). humidus 평균 위치 0.38(도입부), umbrosus 평균 0.84(am 직전). umbrosus: 가장자리 배치율 100%.

**번역 템플릿:**
- 처방문: `[환경?] [부위 BARK→WOOD] [특성 후치] [환경?] am`
- 구조문: `[환경?] [부위 외→내] [특성 삽입] [부위 계속]`

## 6. 주요 발견

### Marci 교차검증 ★★★★★ (v21 신규)
- 2024년 9월 Lisa Fagin Davis가 f1r 다분광 스캔 공개
- Marci 알파벳 대조표(1662~1665): d→d, s→s, t→t, r→r, k→k, l→l = v21 기능어와 직접 일치
- c→sh: sh = suber(/s/ 음가)를 독립적으로 지지
- Marci의 실패 = 글자 음가는 맞혔으나 단어 단위 압축 구조를 몰랐음
- Davis 결론("단순 치환 암호가 아님") = 우리 가설을 간접 지지

### 퍼뮤테이션 검증 ★★★★★ (v21 신규)
- 랜덤 어근 집합 1,000회 vs v21 비교
- 테스트1 (풀 파서): z=7.30σ, p<0.001
- 테스트2 (어근만): z=7.64σ, p<0.001
- 1,000회 중 v21 수준 도달: **0회**
- 순환논리 반론 통계적으로 기각

### 집프 법칙 검증 ★★★★★ (v21 신규)
- α=1.035 (중세 라틴어 기준: 0.94~1.05)
- R²=0.979 (자연언어 기준: 0.95 이상)
- Shannon 엔트로피: 10.148 bits (중세 라틴어 기준: 9.5~11.0)
- Hapax 비율: 67.5% (중세 라틴어 기준: 65~72%)

### Bowern & Lindemann 엔트로피 이상 해소 (v21 신규)
- EVA 문자 h₂=2.312 bits/char (이상 낮음)
- 복합 글리프 h₂=3.148 bits/glyph (중세 라틴어 음절 추정치 2.8~3.5 범위 내)
- 이상의 원인: ch, sh, ai 등 복합 글리프를 개별 문자로 취급한 데서 기인
- 직접 측정: Mondino h₂=2.861, Tacuinum h₂=2.887

### 독립 식물 동정 3개 ★★★★★ (v21 신규)
- **f5v = Malva sylvestris (아욱)**: phlegma ×2 — Dioscorides III.148 "plurimum phlegmatos gignit" ✓
- **f28r = Urtica dioica (쐐기풀)**: pilus ×2 + calidus + nocumentum — Pseudo-Apuleius cap.47 ✓
- **f55r = Rosa canina (장미)**: spina ×2 + nocumentum ×4 + odor ×2 — Dioscorides I.99 ✓

### f99~f102 단면도 도상학 ★★★★★ (v21 신규)
5가지 독립 증거:
1. 색상 코딩: 갈색(f99r)=목질 조직, 초록(f100r)=초본 조직
2. 형태: 동심원 층위, 유기적 외곽선 ≠ 도자기 약통
3. 레이블 내용: cortex, suber, duramen이 v21 파서로 해독
4. 도해 전통: 알베르투스 마그누스, Mondino Anathomia, Carrara Herbal
5. 텍스트 어순: suber(0.369)→cortex(0.485)→medulla(0.531)→duramen(0.571) = Mondino 외→내 순서

### 천문 섹션 계절 달력 ★★★★☆ (v21 신규)
5가지 독립 증거:
1. exterior(seasonal) 천문 섹션에 19.6배 집중
2. 조디악 시계 위치별 계절 구조: χ²=16.5 (가을 집중)
3. Tacuinum Sanitatis 4계절 대응: 봄=cortex/suber, 여름=chylos/radius, 가을=alburnum/duramen, 겨울=cortex/pilus
4. f58v/f58r: alburnum 12.5% (전체 최고) = 가을 채취 달력 텍스트
5. Homo Signorum 선례: Fasciculus Medicinae(1491, 볼로냐-파두아)

### Tacuinum Sanitatis 구조 대응 ★★★★★
- 1380~1450 북이탈리아 (보이니치와 완전 동시기·동지역)
- natura/iuvamentum/nocumentum/remotio/quid generat 표 구조

### Mondino Anathomia 압축 확정 매칭
- cortex→chol, alburnum→al, duramen→daiin, annulus→aiin, suber→shol, radius→ar
