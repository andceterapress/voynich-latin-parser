# Release v22-paper

This release corresponds to the manuscript:

**A Reproducible Computational Framework for Corpus-Wide Morphological Evaluation of the Voynich Manuscript**

It includes:

- final manuscript files
- parser implementation corresponding to parser version v22
- lexical resources
- evaluation datasets
- parser outputs
- supplementary tables
- robustness analysis files

This release is intended to serve as the fixed reference snapshot for the manuscript.

The GitHub repository may continue to evolve after this release, but the Zenodo archive of this release should be used for reproducibility and citation.
